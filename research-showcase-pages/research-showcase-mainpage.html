What is this showcase about?

The guidelines part of these pages deals with the basic theoretical and legal information on querying and researching social media platforms. Here, we want to demonstrate what it actually looks like and how it can be done.
The main goal is to show you how social media platforms (in our case BlueSky) can be queried using their API endpoints. However, we would also like to show you how to combine these with other APIs which could help you with your research and analysis. You will, therefore, find on these pages some demos for using various other API endpoints and a short explanation on how they work.

Our demos

These are our modules you can explore:

  • BlueSky Keyword Search - this one is straight to the point, here you can search the BlueSky social network for posts containing a certain keyword
  • Demographic Predictor - here you can predict a user's (or anyone's, for that matter) age, gender and nationality based on their name
  • Linguistic Post Analysis - use this one to get a CoNLL-U style UD analysis of a sentence, powered by LINDAT
  • The Football Pipeline - all of the above combined in a pipeline based on some football data
