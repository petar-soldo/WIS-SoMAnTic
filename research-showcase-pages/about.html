About SoMAnTic

SoMAnTic (Social Media Analytics Tutorial) is a tutorial-like website aimed at showcasing various approaches to and possibilities of social media analytics.

Project Description

The platform is intended for researchers and curious individuals of varying backgrounds who are interested in analyzing social media posts, particularly from a linguistic perspective using APIs. The website is structured into two major sections: Guidelines and Live Query Showcases.

Motivation

Accessing social media data is crucial for research today, as these platforms serve as an echo chamber of society. Because APIs represent the most direct route, this project focuses heavily on them. Recognizing that researchers may not fully understand what a platform API can offer, we aim to spare them the burdensome request process if it is unnecessary by providing an overview of tools to use platform APIs effectively.
