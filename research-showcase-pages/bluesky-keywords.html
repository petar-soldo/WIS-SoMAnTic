How to query BlueSky posts?

Hello, researcher! In this demo, you can test searching for BlueSky posts, based on keywords you find interesting.

It was made to showcase how querying social media in general could look like. We are using BlueSky API to do it, because it is the most accessible (relatively) widely used social media platform today (a phenomenon you can read about more in our guidlines.) The small application itself is quite simple - you just enter a keyword you would like to find posts about, your BlueSky handle (i.e. your username) and your BlueSky password (you can generate an app password not to use your personal one) and it will return 10 most recent posts from BlueSky which contain that word, together with some metadata.

Scroll down to test how it works!

How does it work?

This small demo works by taking you keyword and posting it to a BlueSky API. To get your results, we actually have to make two API calls. The first API call establishes a connection by using your handle (i.e. username) and password. When we post them, the API sends us an 'authorization token' in return. We use this token in the second API call. We send it to another API point, together with your keyword and in return we get the posts we are interested in. We access the data we need and have the script print out a nice table!

What can I use it for?

This is a very simple and direct content analysis method. Think about events or people you are interested in or whose surrounding discourse you would like to analyze. Just type them in and see what users of BlueSky have to say about it!
